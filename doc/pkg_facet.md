
pkg_facet
=========
Manage Oracle Solaris package facets


Parameters
----------

- **ensure**
    The basic property that the resource should be in.
    Valid values are `present`, `absent`. 

- **name**
    The facet name

- **value**
    The value for the facet
    Values can match `/[tT]rue/`, `/[fF]alse/`, `/[nN]one/`.

Providers
---------
    pkg_facet
